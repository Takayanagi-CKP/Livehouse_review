## 「ライブハウスレビューサイト作ったんだけど口コミある？」

https://kuchikomiaru.com/livehouse_list.php

基本的な機能はだいたい実装できましたが、以下の機能は未実装でこれから実装予定です。
- 連続投稿クッキー対応
- 404ページなど
- 投稿のLine通知
- コメントをDocに変更
- 各種バッチ

    * ライブハウス毎のレビュー数と出演者レビューの集計
    * ライブハウス毎の平均評価算出、その他
- Unit Test
